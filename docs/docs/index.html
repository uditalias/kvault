Introduction

KVault Documentation

KVault is a fast, native desktop client for Cloudflare Workers KV. Browse every namespace across every account, search globally, edit with Monaco, and never touch the dashboard again.

What is KVault?

The Cloudflare dashboard was built for configuration, not for working with data. KVault is purpose-built for KV power users — it brings the speed and convenience of a native desktop app to everything the dashboard makes tedious: cross-namespace search, bulk operations, proper editing, saved sessions, and keyboard-driven navigation.

Built for KV power users

KVault stores API tokens in your OS keychain, caches key names locally in SQLite for instant filtering, fetches values on-demand, and ships with ten built-in themes that apply to both the UI and the Monaco editor. Everything is keyboard-accessible via the command palette.
