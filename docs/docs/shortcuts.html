Keyboard Shortcuts

The complete reference for every keyboard shortcut in KVault. Shortcuts are shown with the macOS Cmd key — on Windows and Linux, replace it with Ctrl.

Global

Navigate the app, open the palette, and jump between namespaces from anywhere.

ShortcutAction
Cmd+KOpen the command palette
Cmd+,Open Settings
Cmd+PQuick open a namespace
Cmd+FFocus the key filter input

Editing

Save your current value or create a new key without leaving the keyboard.

ShortcutAction
Cmd+SSave the current value
Cmd+NCreate a new key

Tabs

Manage the tabbed workspace and recover tabs you closed by mistake.

ShortcutAction
Cmd+WClose the active tab
Cmd+Shift+TReopen the most recently closed tab

Selection & Bulk

Act on multiple keys at once from the key list.

ShortcutAction
Cmd+DeleteDelete the selected keys
Cmd+Shift+EExport the selected keys

Accounts & Sync

Connect new Cloudflare accounts or force a namespace re-sync.

ShortcutAction
Cmd+Shift+AAdd a Cloudflare account
Cmd+Shift+RRe-sync the current namespace
Note. On Windows and Linux, replace Cmd with Ctrl. Every shortcut on this page works identically across platforms.
